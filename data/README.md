示例数据分文件存放于本目录，可用于接口调用或初始化。

- `device_models.json`：设备型号
- `devices.json`：设备
- `device_metric_definitions.json`：指标定义
- `inspection_submits.json`：巡检/数据上报示例
- `metric_ai_analysis.json`：AI 分析示例输出
- `equipment_and_asset_management.json`：资产扩展信息
- `spare_usage_cycles.json`：备件寿命计数
- `energy_consumption.json`：能耗计量
- `oee_stats.json`：OEE 统计
- `maintenance_costs.json`：维护成本
